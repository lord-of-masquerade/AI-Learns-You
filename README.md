# 🧠 AI That Learns You

An adaptive machine learning system that predicts student productivity based on behavioral patterns.

---

## 🚀 Features

- Predicts productivity using ML
- Uses study behavior data (hours, focus, sleep, distractions)
- Interactive UI with Streamlit
- Real-time prediction system
- Scalable into a personal AI assistant

---

## 📊 Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn (Random Forest)
- Streamlit (UI)
- Joblib (Model Saving)
